# ブランチ名表示方法
## 手順
## 1. `~/.bashrc`の編集
```bash
nano ~/.bashrc
```

## 2. コードの追加
- 下記のコードをbashrcに追加
```bash
if [ -f /usr/lib/git-core/git-sh-prompt ]; then
    . /usr/lib/git-core/git-sh-prompt
fi
```

## 3. PS1の変更
```bash
PS1='${debian_chroot:+($debian_chroot)}\[\033[01;32m\]\u@\h\[\033[00m\]:\[\033[01;34m\]\w\[\033[33m\]$(__git_ps1 " (%s)")\[\033[00m\]\$ '
```

## 4. 反映
```bash
source ~/.bashrc
```
